LIST OF FILES AND DESCRIPTION
-----------------------------

CPU.java - Java source file to implement the processor
Memory.java - Java source file to implement the memory
Summary.docx - Summary of the project
readme.txt - text file to assist in running the project

HOW TO COMPILE THE PROJECT
---------------------------

Platform: UNIX

Steps:

1. Compile files: javac *.java

2. Run class files: java CPU "sample program" "timer interrupt value"
		example:	java CPU sample1.txt 30
